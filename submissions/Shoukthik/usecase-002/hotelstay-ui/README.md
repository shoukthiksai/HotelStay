# hotelstay-ui

Angular 22 frontend for HotelStay. See the [main README](../README.md) for setup, architecture,
and how to run the full stack.

## Quick commands

```bash
npm install      # install dependencies
npm start        # dev server at http://localhost:4200 (needs the API on :5168)
ng build         # production build into dist/
```

The API origin is the `API_BASE` constant in [src/app/hotel.service.ts](src/app/hotel.service.ts).
